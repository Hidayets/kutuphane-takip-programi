# Kütüphane Takip Programı

Okullar için geliştirilmiş modern kütüphane yönetim sistemi.

## Özellikler

- ISBN ile otomatik kitap ekleme (Google Books API)
- AI destekli otomatik kategorilendirme
- Öğrenci özet analizi ve puanlama
- Ödünç verme takibi
- QR kod ile hızlı erişim
- Cloudflare Tunnel ile uzaktan erişim
- PDF ve Excel rapor oluşturma
- Otomatik güncelleme
